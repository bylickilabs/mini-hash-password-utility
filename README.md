# Mini Hash & Password Utility

A lightweight Python application with a modern **GUI** that allows you to **encrypt and decrypt text and files**.  
The app includes a **loading screen with ASCII art**, **language selection (EN/DE)**, an **Info button**, and a **GitHub button** linking directly to the developer profile.

---

## ✨ Features

- 🔐 **Text Encryption & Decryption** (AES via `cryptography.fernet`)  
- 📂 **File Encryption & Decryption** (binary-safe)  
- 🌍 **Multi-language Support**: English & German (switch without restart)  
- 🖥️ **Loading Screen** with ASCII Art (`BYLICKILABS`) and progress bar  

---

## 📦 Requirements

- Python **3.8+**
- Dependencies:
  - `tkinter` (comes with Python on most systems)
  - `cryptography`

Install dependencies via:

```bash
pip install cryptography
```

---

## 🚀 How to Run

Clone the repository and run the app:

```bash
git clone https://github.com/bylickilabs/mini-hash-password-utility.git
cd mini-hash-password-utility
python app.py
```
